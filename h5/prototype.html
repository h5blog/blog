js原型链

    var dom = function(){    
    };
    dom.show = function(){
        alert("show");
    };
    dom.prototype.hidden = function(){
        alert("hidden");
    };
 
    dom.hidden(); //error
    dom.show();  
    var d = new dom();
    d.hidden();
    d.show(); //error

1、不使用prototype属性定义的对象方法，是静态方法，只能直接用类名进行调用！另外，此静态方法中无法使用this变量来调用对象其他的属性！

2、使用prototype属性定义的对象方法，是非静态方法，只有在实例化后才能使用！其方法内部可以this来引用对象自身中的其他属性!
