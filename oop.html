面向对象

面向对象三大特点封装、继承、多态

1.封装（类）

将数据和操作数据的方法封装在一起，使其构成一个类。数据被保护在类的内部，可以通过类对外提供的接口来访问类里面的数据

		// 把抽象的事物封装成类
		let Animal = class {
		    constructor() {
		        this.type = 'animal';
		    }
		    getType(){
			    console.log(this.type);
		    }
		    eat(){
		        console.log('动物吃食物');
		    }
		}

2.继承

通过 extends 实现类的继承。父类(Animal)中的属性，子类的实例化对象(tiger)也可以访问

注意类的实例化必须通过new关键字，关于类和实例化对象之间的继承请参照js原型链

		class Tiger extends Animal {}
		var tiger=new Tiger();
		tiger.getType();//'animal'

3.多态

父类中的属性，子类在继承的基础上可以拓展自己的特性。

		class Tiger extends Animal {}

		class Cat extends Animal { 
		    eat(){
			    console.log('猫吃鱼');
			}
		}

		class Dog extends Animal { 
		    eat(){
			    console.log('狗吃骨头');
			}
		}

		var tiger=new Tiger();
		var cat=new Cat();
		var dog=new Dog();
		tiger.eat();//动物吃食物
		cat.eat();//猫吃鱼
		dog.eat();//狗吃骨头
